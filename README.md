# MediaDownloader

```txt
SNS Media downloader.
Version 1.0.0

Target SNS: Twitter

Syntax:   mdl [options] [url ...]
Options:
 -h, --help     Print this message.
 -q             No output log.
 -d DIR         Save download directory. Default ./media
 -v, --version  Print version.
```

## Develop

### TypeScript

```sh
npm i -g typescript
```

of

```sh
npm i typescript
```

### Install module

```sh
npm i
```

### Build

```sh
npm run build
```

or

```sh
tsc -p src
```
